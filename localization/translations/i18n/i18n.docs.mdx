## Component usage

### Translations();

```
import {Translations} from '@enlear/react.i18n';

Translations();
```

### ChangeLanguage(lang: any);


```
import {ChangeLanguage} from '@enlear/react.i18n';

ChangeLanguage(language);
```

### UseTranslation();
```
import {UseTranslation} from '@enlear/react.i18n';

const i18n = UseTranslation();
```
